# v0.6.0 — 2026-03-15

## Features

### Calls (Voice & Video)
- Voice and video call support via Telegram MTProto protocol
- Diffie-Hellman key exchange for E2E encryption
- Incoming call dialog with accept/decline
- Outgoing call screen with ringing state
- In-call screen with mute, volume, duration timer
- Video call screen with local/remote video placeholders
- Group call support (create, join, leave, mute)
- VoIP sidecar process for audio transport (stub)
- Call state management (callStore, groupCallStore)

### Message Forwarding
- Forward messages via context menu or selection bar
- ForwardDialog with chat picker, search, and avatars
- Backend `forward_messages` Tauri command using grammers API

### Avatar Viewer
- Click avatar in ChatInfoPanel to view full-size profile photos
- Navigate all user photos with left/right arrows
- Keyboard navigation (ArrowLeft/ArrowRight/Escape)
- Zoom with mouse wheel, double-click toggle, drag-to-pan
- "1 of N" counter display
- Backend `get_user_photos` command to fetch all profile photos

### Native Notifications
- OS-native notifications for incoming messages (macOS, Windows, Linux)
- Smart filtering: no notification if chat is open, window focused, or chat muted
- Uses tauri-plugin-notification

### Folder Context Menu
- Right-click on folder tabs for context menu
- "Read All" — mark all unread chats in folder as read
- "Mute All" — mute all chats in folder
- "Edit Folder" / "Add Chats" / "Delete" for custom folders

### Message Context Menu & Selection
- Right-click context menu: Reply, Forward, Select, Copy, Pin, Edit, Delete
- Multi-select mode with selection bar (copy, forward, delete)
- Ctrl+C / Cmd+C to copy selected messages
- Markdown rendering toggle per message

### Chat Sorting
- All folders except "All Chats": unread messages sorted to top
- Preserves relative date order within unread/read groups

## Improvements
- ImageViewer: fullscreen mode, zoom, pan, download button
- Merged consecutive messages from same sender into groups
- Theme-aware CSS for ForwardDialog and AvatarViewer
- i18n keys for all new features (en/ru)
- Call debug logging (frontend + backend)
- Sidebar scroll fix
